【はじめに】
 ･このソフトウェアはキーボードからの入力を解析し、キーごとに押された回数を保持します。

【動作環境】
 ･Windows 7,10で動作する事を確認しています。


【インストール】
 ･ファイルをダウンロード後、エクスプローラー等で解凍してから実行ファイル
（KeyborCounter）をダブルクリックしてください。


【アンインストール】
 ･レジストリ等はいじっておりませんので
  インストールしたファイルをフォルダごと削除すれば完了です。
 ･スタートアップに登録した場合はそちらも削除してください。


【使い方】
 ･KeybordCounter.exe を実行すれば自動的にタイプ数とクリック数のカウントを開始します。
　ファイルメニューからリセットすることが可能です。
